# 总结

虽然现在已经不在从事 Android 相关的学习、工作,但是曾经这个不起眼、拿不出手的小项目,却给我以后的学习带来了很多益处.

通过写(抄)这个项目,我从0开始经历了一个产品(项目)的诞生,对UI、 数据库、网络、开源库、如何写代码等等等,有了自己的见解和收获,锻炼了我的自学能力, 也让我知道了英语在编程世界的重要性,可以说现在的一切都起源于这个自学 Android 的小项目.
